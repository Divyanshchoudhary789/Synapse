# 🌐 Synapse - Professional Networking Platform

**Synapse** is a full-stack, professional networking application designed to bridge the gap between opportunities and talent. Built with the **MERN Stack** and **Next.js**, it offers a seamless, high-performance experience for professionals to connect, share insights, and build their digital presence.

---

## 🚀 Live Demo & Deployment
- **Frontend:** https://synapse-s4g1.onrender.com
- **Backend:** https://synapse-backend-ujiu.onrender.com

---

## ✨ Key Features

### 👤 Professional Identity
- **Custom Profiles:** Build a professional persona with detailed info, education, and skills.
- **Dynamic User Cards:** Quick-view profiles to understand professional backgrounds at a glance.

### 📝 Content & Engagement
- **Multimedia Posts:** Share perspectives using high-quality images and videos.
- **Interactive Feed:** Real-time **Like** and **Comment** system to engage with your network.
- **Smart Text Handling:** Support for long-form content with "See More" functionality for better readability.

### 🤝 Networking & Connections
- **Connect System:** Build a meaningful network by connecting with other professionals.
- **Global Search:** Discover and find peers or mentors within the platform.

### 🛠 Technical Excellence
- **Optimized Storage:** Utilizing **Cloudflare R2** for lightning-fast and scalable media storage.
- **State Management:** Powered by **Redux Toolkit** for predictable and efficient data flow.
- **Responsive UI:** Fully responsive design built with **Tailwind CSS** for a premium experience on mobile and desktop.
- **RESTful Architecture:** Clean and scalable API design for robust frontend-backend communication.

| Layer | Technology |
